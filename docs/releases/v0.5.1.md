# Pi-Vis v0.5.1

## Highlights

- Improved reliability across session startup, switching, reattachment, and renderer reloads.
- Added automatic recovery for unexpected renderer crashes, with bounded reload behavior and diagnostics logging.
- Improved project-trust flows, including safer startup prompts and validation of persisted trust choices.
- Improved Bash command presentation so `!` and `!!` output is recorded consistently in the transcript.
- Refined session search, app pickers, extension dialogs, sidebar state, and fullscreen/window messaging.

## Fixes and polish

- Fixed lifecycle races that could leave sessions, panels, or renderer state out of sync.
- Prevented window notifications and teardown paths from crashing during window destruction.
- Improved command admission and state reconciliation around trust, Bash execution, and session activation.
- Expanded coverage for crash recovery, session registry/search, picker and dialog behavior, protocol state, and real SDK-host flows.

## Upgrade notes

- No manual action required. Existing sessions and saved state continue to work normally.

## Assets

- macOS Apple Silicon ZIP for installer/auto-update
- macOS Apple Silicon DMG for manual install
