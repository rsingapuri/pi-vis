# Pi-Vis v0.4.3

## Highlights

- Added inline code review comments in the diff viewer:
  - Click diff lines to compose targeted comments without leaving the review.
  - Queued comments are preserved with the session and can be inserted into the composer.
  - Diff sections now have clearer hover, selection, and comment affordances.
- Improved model selection for accounts with many providers:
  - Models can be grouped by provider in the session header picker.
  - Added a compact transcript display option in Settings.
  - Keyboard navigation works across provider groups and filtered model results.
- Improved terminal and custom UI keyboard handling:
  - Added Kitty keyboard protocol negotiation for unified TUI panels.
  - Modifier keys, Shift+Enter, ESC, and terminal focus behavior are handled more consistently.
  - Custom panels close more reliably without leaving extension UI state behind.

## Fixes and polish

- Fixed extension/custom panel close edge cases where the host could miss or duplicate close handling.
- Fixed sidebar interaction edge cases and added coverage for expanded/collapsed workspace behavior.
- Improved transcript spacing in compact and default display modes.
- Tightened image lightbox layout so large images fit the viewport more cleanly.
- Improved diff rendering styles, comments, and empty/edge states.
- Improved login terminal and preview stub behavior for keyboard/event handling.

## Tests and tooling

- Added unit and E2E coverage for diff comments, model/provider grouping, compact transcript mode, sidebar interactions, and extension panel close behavior.
- Added real Kitty keyboard protocol E2E coverage for unified panels.
- Updated CI/release documentation for the expanded verification flow.

## Upgrade notes

- No manual action required.
