# Pi-Vis v0.4.6

## Highlights

- Improved steering and queued-message workflows:
  - Queue prompts while a session is busy and manage queued messages more reliably.
  - Improve steering behavior and keep renderer, host, and session state synchronized through interruptions and reloads.
- Improved the unified TUI behavior with more reliable input, focus, and panel behavior.
- Added a clear collapse/expand affordance for compact transcript history, making long conversations easier to scan.
- Improved diff viewer refresh behavior and Escape handling in diff and branch controls.

## Fixes and polish

- Fixed composer focus and autocomplete interactions, including focus restoration after command completion.
- Fixed session and composer state restoration after reloads.
- Improved archive confirmation warnings and sidebar archive interactions.
- Refined transcript, session header, extension panel, picker, notification, and worktree-control styling and spacing.
- Fixed small UI inconsistencies across compact mode, diff editing, branch selection, and custom panels.

## Runtime and testing

- Expanded coverage for steering, queued messages, reload recovery, unified TUI behavior, transcript history, diff interactions, archive workflows, and composer autocomplete.

## Upgrade notes

- No manual action required. Existing sessions and saved state continue to work normally.

## Assets

- macOS Apple Silicon ZIP for installer/auto-update
- macOS Apple Silicon DMG for manual install
